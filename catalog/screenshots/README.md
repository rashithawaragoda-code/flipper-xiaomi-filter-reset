# Screenshots

Catalog screenshots must be captured from a real Flipper Zero using qFlipper's
screenshot feature (128x64, 1-bit), and must not be resized or re-encoded.

Place at least one here (e.g. `screenshot_1.png`) and reference them from
`catalog/manifest.yml` before submitting to the Flipper Application Catalog.

Suggested shots:
1. The start menu.
2. The "hold the tag" reset screen.
3. The success screen.
